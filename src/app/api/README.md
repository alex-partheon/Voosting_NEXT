# API Routes

이 폴더는 Next.js App Router의 API 라우트를 포함합니다.

## 주요 API

- `/api/auth/*` - 인증 관련 API
- `/api/campaigns/*` - 캠페인 관련 API
- `/api/users/*` - 사용자 관련 API
- `/api/payments/*` - 결제 관련 API
- `/api/referrals/*` - 추천 시스템 API

## 규칙

- RESTful API 설계 원칙 준수
- Supabase Edge Functions 활용
- 에러 처리 및 로깅 필수
