## 테스트 진행 방법

### 1. 개발 서버 실행

```bash
npm run dev
```

### 2. 브라우저에서 접속

- URL: http://localhost:3002
- 로그인 페이지로 이동: http://localhost:3002/auth/signin

### 3. 각 계정으로 순차 테스트

1. 크리에이터 계정으로 로그인
2. 기능 테스트 후 로그아웃
3. 비즈니스 계정으로 로그인
4. 기능 테스트 후 로그아웃
5. 관리자 계정으로 로그인
6. 기능 테스트 완료
